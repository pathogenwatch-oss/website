Whole Genome Sequence Analysis (WGSA)

WGSA.net aims to provide easy to use interpretation tools for the analysis of whole microbial genome sequence datasets.

Features

Tree Viewer

  • View Genomic clustering of isolate sequences.
  • Overlay metadata.
  • Highlight branches.

Data

  • Containing results of resistance predictions and MLST.
  • Merge Collections together.

Maps

  • Geographic origin of isolates.
